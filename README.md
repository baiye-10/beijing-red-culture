# 探寻北京中轴线红色文化足迹 - 网页项目

## 项目简介

本项目是基于"文科项目最终成果"PPT内容制作的互动式网站，以北京中轴线红色文化为主题，包含意象、漫游、行走三大板块，展示中轴线历史文化风貌和红色足迹。

## 主要功能

### 主页面
- **首页展示**：项目主题和概述
- **意象板块**：展示中轴线文化意象
- **漫游板块**：叙事地图（五四运动、一二九运动）
- **行走板块**：5条漫步北京路线

### 叙事地图子页面
- **五四运动**：历史介绍、路线地图、景点详情
- **一二九运动**：历史介绍、路线地图、景点详情

### 行走路线子页面
- 路线一：天安门·国家博物馆·人民大会堂
- 路线二：景山·北海·故宫角楼
- 路线三：胡同·什刹海·钟鼓楼
- 路线四：天坛·先农坛·永定门
- 路线五：奥林匹克公园·中轴线北延

### 景点详情页面
- 景点介绍
- 游客评价情感分析展示
- 高德地图位置显示

## 使用说明

1. 将PPT中的图片导出到 `images/` 目录
2. 申请高德地图API Key，替换HTML文件中的 `YOUR_AMAP_KEY`
3. 使用浏览器直接打开 `index.html` 即可查看网站
4. 推荐使用Live Server等工具进行本地预览

## 设计风格

- **配色方案**：红色（主色）、深蓝（辅助色）、金色（点缀色）、米色（背景）
- **字体**：Noto Serif SC（古风）+ Noto Sans SC（现代）
- **设计理念**：古风与现代结合，传承红色文化
