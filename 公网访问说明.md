# 网页公网访问配置指南

## 方式一：使用 ngrok（推荐，最简单）

### 1. 下载并安装 ngrok
- 访问：https://ngrok.com/download
- 下载 Windows 版本
- 解压到任意目录

### 2. 注册账号并获取 Authtoken
- 注册 ngrok 账号：https://ngrok.com/signup
- 登录后获取 Authtoken：https://dashboard.ngrok.com/get-started/your-authtoken

### 3. 配置 ngrok
```bash
# 在 ngrok 解压目录下运行
ngrok config add-authtoken 你的authtoken
```

### 4. 启动内网穿透
```bash
# 在 ngrok 解压目录下运行
ngrok http 8000
```

### 5. 获取公网访问地址
启动后会显示类似：
```
Forwarding  https://xxx-xxx-xxx-xxx.ngrok-free.app -> http://localhost:8000
```
将这个 HTTPS 地址分享给其他人即可访问！

## 方式二：使用花生壳（国内访问更快）

### 1. 下载花生壳
- 访问：https://hsk.oray.com/download/
- 下载 Windows 客户端

### 2. 注册账号并登录
- 注册花生壳账号
- 登录客户端

### 3. 配置内网穿透
- 点击"内网穿透"
- 添加映射：
  - 应用名称：北京中轴线红色文化
  - 内网主机：127.0.0.1
  - 内网端口：8000
  - 外网端口：自动分配

### 4. 获取公网地址
启动后会分配一个公网访问地址

## 注意事项

1. **本地服务器必须保持运行**
2. **ngrok 免费版每次重启地址会变化**
3. **建议使用 Chrome/Edge 浏览器访问**
4. **百度地图 API 需要在公网环境下才能正常显示**
